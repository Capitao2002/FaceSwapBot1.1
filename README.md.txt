FaceFusion Telegram Bot
Troca rostos de vídeos com IA (via FaceFusion) em um bot no Telegram.

Como usar
Envie um vídeo .mp4 para o bot
Envie uma imagem com o rosto desejado
O bot responderá com o vídeo modificado
Deploy no Render
Crie um novo Web Service
Use este repositório (com Dockerfile)
Start Command: deixe em branco
Instale Python 3.10+
Configure seu TELEGRAM_TOKEN em bot_config.py
Pronto! O bot estará funcionando publicamente via Render.
